华硕新平板电脑现场评测
site name  2013-11-04
原文地址：http://pad.pconline.com.cn/332/3328549.html

　　【PConline 现场报道】作为2013台北国际电脑展的东道主之一，华硕的表现自然不会领我们失望。就在电脑展开幕的前夜，华硕以“WE TRANSFORM”为主题召开了全球新品发布会，并带来了多款极具创意的电脑和平板产品。其中Fonepad Note FHD 6和MeMO Pad HD 7两款平板产品颇具看点，从现场演示以及产品配置来看，这两款平板可以说是专门针对三星Galaxy Note系列而推出的。下面小编就带各位现场评测一下这两款平板电脑。

华硕Fonepad Note FHD

　　首先评测的是Fonepad Note FHD 6。这是一款6寸手机平板，轻薄小巧的机身单手便可完全掌控。Fonepad Note FHD 6的配置十分强悍，屏幕方面，Fonepad Note FHD 6采用Full HD1920×1080超高分辨率显示幕，材质为Super IPS+。这款屏幕不仅具有超广视角，其亮度表现也相当优异，最高亮度可达450 nits，无论室内外均能获得极佳的显示效果。

　　另外，Fonepad Note FHD 6也是华硕第二款采用Intel Atom处理器的手机平板，相较于上一代的Fonepad，这款平板的处理器升级到了双核Z2560，主频最高可达1.6 GHz，性能表现将更加出色。

　　与三星Galaxy Note相类似的是，华硕Fonepad Note FHD 6也配备有触控笔，这将带来更加精准的触控操作。结合上内置的记事应用程，用户输入文字和记录创意也更加便捷。

　　Fonepad Note FHD 6也延续了华硕一贯出色的影音特色，前置双扬声器搭配上独家的SonicMaster声音技术，将带来相当出众的影音体验。

　　通话方面，Fonepad Note FHD 6也集成了前代Fonepad双麦克风设计，通话效果十分出色。

华硕MeMO Pad HD 7

　　华硕MeMO Pad HD 7延续了MeMo系列一贯炫彩的外观，具有白、绿、黑及桃红四种机身颜色，可以满足时尚青年对于个性的追求。不仅如此，MeMO Pad HD 7的机身也十分轻巧，仅302克的重量让随身携带毫无负担。

　　另外，MeMO Pad HD 7的配置也很不错，搭载了一颗A7架构的四核处理器，操作极为流畅。

　　显示方面，MeMO Pad HD 7配备了一块7采用1280×800 高清IPS屏幕，这块屏幕不仅显示效果极其出色，更具有170度超广视角。另外，MeMO Pad HD 7还配备120万像素前置摄像头及500万像素后置摄像头，并支持720P录像功能。

　　同时，MeMO Pad HD 7也支持华硕独家的SonicMaster声音技术，配合上双扬声器，将带来十分出众的影音享受。

　　另外，华硕还为MeMO Pad HD 7定制了专用保护套，保护套同样有丰富多彩的颜色可供选择。

 　　从现场的试玩来看，Fonepad Note FHD 6的定位可以说是比照着三星Note系列而设定的。与华硕上一代通话平板Fonepad相比，Fonepad Note FHD 6升级幅度相当大，不仅搭配了英特尔双核处理器，屏幕也提升至了1080P全高清水平。更为特别的是，Fonepad Note FHD 6还引入了触控笔功能，这与三星的Galaxy Note系列有着异曲同工之妙，操控性和文字录入速度都有了很大的提升。MeMO Pad HD 7延续了MeMO系列一贯青春靓丽的风格，多彩的机身很吸引眼球。同时MeMO Pad HD 7的硬件也有着大幅度提升，不仅搭载了四核处理器，屏幕也升级至了720P高清水平。配合上华硕这个品牌，这两款产品的市场前景应该会非常不错。
