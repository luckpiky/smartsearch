■ 8寸大屏幕引入窄边框设计

　　外形方面，Galaxy Tab 3 8.0与前代产品相比并没有太大的创新，依然保是保持三星经典的鹅卵石造型。不过，由于Galaxy Tab 3 8.0引入了现今最流行的窄边框设计，不仅看上去十分具有冲击力，整机也变得更加迷你，配合上仅7.4mm机身厚度和314g的机身重量，Galaxy Tab 3 8.0更加易于随身携带。机身背面则保持了一贯简洁的风格，除了一枚500万像素摄像头和LOGO之外，没有任何多余的设计，另外侧边的银灰色边框则为整机提分不少。


机身正面


机身背面

　　Galaxy Tab 3 8.0机身整体布局同前代产品也十分类似，屏幕上方的两个小圆点分别为前置130万像素摄像头和光线感应器，由于不是3G版所以并没有听筒。而下方则是Android必备的“三大天王”按键，其中实体Home键保持了三星Galaxy家族一贯的特色，不过与此前的产品相比，这颗Home键要显得更加纤细一些。值得注意的一点是，Home键两侧的两枚虚拟按键按触时并没有震动反馈，不过用户可以选择触摸提示音，此外还可以选择按键是否常亮。


前置摄像头和按键

　　材质方面，Galaxy Tab 3 8.0依然采用了此前饱受诟病的工程塑料。不过将Galaxy Tab 3 8.0拿在手里时，小编感觉不到它有丝毫的廉价感，无论是极窄边框的设计，还是7.4mm的超薄机身以及314g的极致重量，无不体现出了三星在工业设计方面的超凡造诣。至于手感，这是一个见仁见智的东西，有人偏好金属的冰冷坚硬，有人则喜欢塑料的温暖柔滑。


单手可轻松握持

■ 机身细节设计人性化

　　同三星大屏手机一样，Galaxy Tab 3 8.0的电源键和音量键也都设计在了机身的右侧，这种设计十分适合单手操作，无论是左手还是右手握持时使用起来都非常方便。另外，在机身右侧还可以看到一个红外接口，借助这个红外接口和内置的遥控应用，只需经过简单的设置，Galaxy Tab 3 8.0就能摇身一变成为一个万能电视遥控器。


电源及音量键

　　Galaxy Tab 3 8.0的内存卡槽位于机身的左侧，并配备有防尘盖。而机身扬声器和数据接口的设计，则明显是借鉴了此前的Galaxy Note 8.0，同样是位于机身的底部，这样的设计有两个好处，一方面充电时也可以方便用户使用，另外播放视频或音乐时，扬声器也不宜于遮挡，以至于影响声音效果。此外，耳机接口则被安放在了机身顶部，使用起来也十分方便。


机身细节

　　人性化，这是小编对Galaxy Tab 3 8.0机身细节设计最深的感触。就拿电源键举例吧，市面上的7寸或者8寸平板的电源键机身上都是位于机身的顶部，当我们单手操作的时候，基本上很触碰到，所以当我们点亮或关闭屏幕的时候，往往都需要双手操作。而Galaxy Tab 3 8.0将电源键设计在了机身右侧，则很好的解决了这个问题，这充分体现出了三星以人为本的设计理念。

■ 主流的硬件性能

　　硬件方面，三星Galaxy Tab 3 8.0并没有过于超前，仅搭载了一颗1.5GHz双核处理器，和1.5GB系统内存以及8GB的存储空间。此外，Galaxy Tab 3 8.0还拥有一块8.0英寸大屏幕，分辨率为1280×800像素，屏幕比例为16：10。一下为安兔兔的测试结果。

■ 系统及操作初体验

　　Galaxy Tab 3 8.0搭载了最新的Android4.2.2操作系统，并内置了三星独家的TouchWiz操作界面。经过几年的的发展，三星这套TouchWiz界面已经十分成熟了，不仅集成了来电黑名单、多功能通栏、省电优化等贴心小功能，还具备有浮动视频播放、多任务界面三星等独有功能。下面，就请各位同小编一同体验一下吧。

　　相对于原生的Android系统来说，三星Galaxy Tab 3 8.0所搭载的系统在视觉方面更加的明快绚丽、尤其是新的锁屏界面很炫目，不仅可以添加各种小插件，还可以显示用户的定制信息。另外，三星所有设备均采用了统一的TouchWiz界面，这也就免去了用户更换设备时的学习时间，一拿到设备就能直接上手使用，同时还能提升了用户对三星的品牌忠诚度。

■ 续航发热

　　三星Galaxy Tab 3 8.0内置电池容量为4450毫安时，这与动辄七八千毫安时的国产平板相比并没有任何的优势，但是三星在硬件设计以及系统优化方面的实力并不是国产平板厂商所能比拟对的，由于Galaxy Tab 3 8.0所搭载的硬件功耗并不是非常强悍，且内置了完备的电源管理系统，所以Galaxy Tab 3 8.0的续航能力还是很令人满意的。经过测试，充满电后Galaxy Tab 3 8.0无线网络连接状态下纯待机3天，仅耗电7%左右，连续播放高清视频3个小时，耗电量为40%，也就是正常使用情况下，Galaxy Tab 3 8.0坚持一天是完全没有问题的。另外，Galaxy Tab 3 8.0的发热量也控制的很好，连续使用状态下，机身感觉不到明显的发热。

■ PConline评测室总结

　　三星Galaxy Tab 3 8.0的总体表现令小编十分惊艳，其不愧是“名门之后”，完全继承了三星Galaxy系列的优良血统，无论是圆滑的鹅卵石造型，还是方便易用的TouchWiz操作界面，无不彰显着三星高超的工业设计水平和人性化的设计理念。此次，三星Galaxy Tab 3 8.0还引入的极窄边框和超薄机身设计，不仅令机身更具冲击力，将便携性提升至了更高水准。而且，三星Galaxy Tab 3 8.0这次的定价更加亲民化，仅2288元的起步价格相信会引爆今年的夏日平板市场。
