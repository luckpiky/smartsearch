千元清新时尚派 三星Tab 3 7.0平板评测
手机中国  2013-11-10
原文地址：http://pad.cnmo.com/29/296495.html

　　【手机中国 平板】相信这一段时间大家在很多电视广告上都看到了吴莫愁代言的三星Tab 3系列平板，其实这款平板是有三个尺寸的版本，其中8.0版本是其中最为纤薄的一款，而7.0版本则是这个系列中尺寸最小最易携带的一款。

　　作为这个系列中我们最后评测的产品，这款三星Tab 3 7.0版和其他两个版本除了产品在产品外观上有很大的不同，但在UI方面则是一致的，这也保证了三星产品的统一性。

　　作为今年三星推出新品中最为实惠的一款平板，三星Tab 3 7.0在外观上十分吸引人，虽然性能上有些许的不足，但外观上的做工还是为这款产品赢回不少的掌声，那么这款产品的表现究竟如何呢？下面就让笔者为大家带来这款产品的评测。

圆润精细的外观

　　三星Galaxy Tab 3 7.0的机身三围尺寸为188×111.1×9.9毫米，重量仅为304克。这个重量在7英寸平板中算是轻量级的，因此即便是长时间的握持也不会觉得太累，但遗憾的是在机身的厚度上没有Tab 3 8.0版本控制的那么好。


 三星Tab 3 7.0

　　三星Tab 3 7.0版的正面设计延续了三星今年整体的设计理念，不管是在边框的处理上还是实体按键的设计上都延续了起一贯的风格，但在四周边角的处理上要相对更加圆润一些。


 三星Tab 3 7.0背面设计

　　背部的的设计依旧是这样，三星的“塑料”质感依旧延续着，手感握着很为顺手，但质感不足，不知道三星什么时候可以摆脱这个魔咒。


 三星Tab 3 7.0包边设计

　　边框的设计很为讨巧，在四周的边角有着非常不错的弧度设计，这样一方面可以在外观上看起来很为顺畅，同时在手感上也更为出色，给人的感觉非常舒服，虽然边框的材质依旧是塑料，但泛着金属的光泽。

出色的窄边框设计

　　三星Galaxy Tab 3 7.0的整体设计还是颇为简洁的，相比去年的Tab 2系列，三星在这代产品上加入了赤手火热的“窄边框”设计，这样的设计在8.0版本和10.1版本上都有体现。

　
三星Galaxy Tab 3 7.0机身按键

　　三星Galaxy Tab 3 7.0的侧面按键设计的和边框一致，但按键整体键程较短，手感方面比较一般。

　
三星Galaxy Tab 3 7.0机身细节

　　在机身的细节上，三星Galaxy Tab 3 7.0和另外两个版本的设计没有任何的不同，同样是实体的“Home”按键，底部的两个双扬声器还是十分给力的，同时机身的底部和侧面分别有着一个麦克风。


三星Galaxy Tab 3 7.0对比iPad mini

　　由于加入了窄边框的设计，因此笔者也拿来了目前7英寸平板最有代表性的产品iPad mini来和这款平板进行对比。


三星Galaxy Tab 3 7.0对比iPad mini边框

　　三星Galaxy Tab 3 7.0和iPad mini的机身比例不成正比，但在边框设计上iPad mini可是先胜一筹，我们也希望更多的Android平板能在边框设计上下些功夫，这样屏幕的显示面积会更大，视野也会更好。

性能跑分测试

　　三星Galaxy Tab 3 7.0采用了采用7英寸1024×600分辨率触摸屏，300万像素后置摄像头和130万像素前置摄像头。其内置Marvell PXA988双核处理器，处理频率为1.2Ghz，PXA988这款处理器专为中国市场而设计，拥有1080p编解码能力，使之能够适合高品质多媒体播放和视频捕捉。除此之外，PXA988还配备了集成图像信号处理器（ISP）和图形处理单元，其采用GPU具体型号为Vivante GC1000。这款处理器的性能究竟如何呢？下面就让我们来看一下。

　
安兔兔跑分测试

　　安兔兔跑分方面得到了12121分的成绩，这个得分已经比肩一些四核处理器了，因此这款处理的性能还是值得肯定的。

　
HTML 5测试和象限测试

　　在HTML 5和象限测试方面这款平板的得分也是十分不错的，HTML 5的分值属于中等，而象限方面的的3827比三星Galaxy Tab 3 8.0的得分还要高些。


NenaMark 2测试

　　总体看来，这款处理器的性能还是十分不错，但由于内存的限制，性能整体提升的空间还是十分有限，但在体验上这款平板还是十分不错，一点也不会让人看出是双核平板。

深度定制的UI

　　三星Galaxy Tab 3 7.0在系统方面采用最新的TouchWiz 5.0界面，从锁屏到机身内部的图标都是三星统一的风格，在下拉菜单、底部状态栏方面也都进行了重新的设计。整体的色调和机身更加的融合。

　
最新的TouchWiz 5.0

　
解锁及下拉菜单

　　解锁画面和8.0版本有很大的不同，解锁的特性进行了取消，但看起来更加的实用。

　
后台及小组件

　　系统方面也只是采用了Android 4.1.2，并没有采用三星Galaxy Tab 3 8.0上的Android 4.2系统。总的来说三星Galaxy Tab 3 7.0上的UI是三星Galaxy Tab 3 8.0的一个简化版，特性减少更为实用。

拍照样张解析

　　三星Galaxy Tab 3 7.0后置了300万像素的摄像头，但遗憾的是这颗摄像头不支持自动对焦，因此在一些画面的处理上还是有很大的不足，但究竟如何？让我们来看一下三星Galaxy Tab 3 7.0的样张。

　　总体来说照片的的质量不高，但颜色的还原还是不错，在细节方面少了自动对焦，细节缺失不足。

影音体验及最后总结

　　影音播放时目前平板电脑上最为主要的功能，笔者也是通过一系列的测试来看看这款平板的视频播放究竟如何。经过测试，这款平板可以完美的支持全格式的720p视频播放。1080p的视频播放也没有问题，但个别格式不支持，可以通过下载第三方播放器进行解决。

　　在视频播放时，三星Galaxy Tab 3 7.0还有一个不错的功能，那就是在播放时，按一下锁屏按键可以进行按键锁屏，视频正常播放，但其他按键不管如何触碰都不会造成误操作。

　　总结：作为三星的一款入门级平板，Galaxy Tab 3 7.0的推出很好的继承了P3100的地位，在性价比上更为突出，目前这款平板8G Wi-Fi版在京东上的报价为1459元，虽然是三星平板中最为实惠的一款，但对于普通消费者来着价格还是略显昂贵，想要购买的朋友不妨再等一等。
