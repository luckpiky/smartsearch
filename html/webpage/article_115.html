X86+酷睿芯才够劲！FHD版乐凡F2平板评测
太平洋电脑网  2013-10-19

　　【PConline 评测】当初Windows 8登陆平板时无数小伙伴为之喝彩，因为平板很可能因为Windows 8的到来从而甩掉“玩具”的称号。确实，在iPad与Android平板占主导地位的时代，想要用平板干活基本上是不现实的事，而在Windows 8上则可轻松拿下。Atom+Windows 8的组合或多或少有当年昙花一现的上网本的影子，在若干网友评论中我们看到了Core+Windows 8的平板组合更受青睐，加一个键盘则更加完美。


乐凡 F2钻石版(64G/3G版)  图片  系列  评测  论坛  报价  网购实价

　　在众多跟进Windows 8平板的厂商中，乐凡是少有的国产品牌之一。据悉，乐凡一直致力于Windows平板，此次的F2是乐凡联手富士康之作，工艺品质再次提高。与此同时，F2拥有2999元到5599元多款不同配置，更有高端定制版，相比其他Windows 8平板而言可选的余地更大。

11.6英寸FHD屏双刃剑

　　本次我们拿到的是乐凡F2钻石版之一，搭载Intel Core i5-3339Y处理器，配备4GB内存和64GB存储空间，支持WCMDA制式3G模块，售价4999元，属于乐凡F2系列的中高配备。除了出色的硬件配备外，11.6英寸的乐凡F2还拥有一块1920×1080全高清IPS屏幕，在拥有出色性能的前提下显示效果也告别了万年不变的1366×768，相信这两点会给用户留下更好的印象。

　　乐凡在外包装盒上也下了不少心思，各种产品特色都印制在五颜六色的Windows UI“豆腐块”上，好看的同时也表明了Windows 8的特质，可谓一举两得。


Windows 8风格包装盒

　　最近笔者把玩的都是8英寸的小家伙，突见11.6英寸的乐凡F2时视觉冲击很大，11.6英寸大屏幕视觉体验更宽广，特别是在办公时不会感觉屏幕大小捉襟见肘。当然，选择大屏幕时就必须要对便携性做一定妥协了，操作乐凡F2时必须双手同时进行。


11.6英寸FHD屏

　　正面的配备比较常见，顶端的200万像素摄像头为视频应用提供了便利，底端的Windows虚拟按键也是Windows 8产品的标配。相比虚拟按键，笔者倒是更喜欢实体按键，按着踏实。


Windows触摸键

　　乐凡F2的IPS屏幕拥有出色的可视角度，亮度达到了350nits，在阳光和反光环境下也能带给用户较好的视觉体验。客观来说，FHD标准的确很酷，不过在Windows 8系统上是把双刃剑，Windows UI界面下显示效果细腻，切换到传统界面后在100%字体大小时长时间看屏幕的话眼睛还是会有一定的酸痛。

四风口跟热量说再见

　　乐凡F2的背面采用了白色设计，清新时尚。多处孔状很显眼，底端左右两侧各有一个扬声器，剩下的四个位置均是散热孔。其中，左右两侧为进风口，顶端两个则是出风口。目前搭载Intel Core处理器的平板都会设计有出风口，因为强劲处理器带来高性能的同时发热也会增加，多散热口的加入会让整机尽量保持在一个较为舒适的使用环境下。

顶端出风口

　　一般双手握持机器时，热风从顶部排出，不会影响到用户，整体来看，乐凡F2在这方面的设计还是很合理的。

双全尺寸USB接口不冲突

　　乐凡F2的厚度为13.5mm，重量为850g，这一参数和Surface Pro相当。机器定位的原因不能让它和那些所谓的全球最轻薄平板相提并论，不过在同类产品中乐凡F2的表现还是不错的。难能可贵是，乐凡F2在保证机身厚度的同时没有像接口妥协。


左侧USB 3.0接口

　　机身左侧依次安排有耳机插孔、USB 3.0接口、Mini HDMI接口和电源接口，右侧则是一个USB 2.0接口和Micro SD卡槽。


右侧USB 2.0接口

　　顶端左侧开关旁边是3G SIM卡槽和重力感应开关。两个全尺寸USB接口绝对是良心之作，因为像这类平板你不可避免的要用其中一个USB接口插无线键鼠接收器，另一个就可以完成接驳U盘等外设的工作。倘若机身只有一个USB接口，要完成上述操作的话肯定就会撞车。


顶端SIM卡槽|重力感应开关

　　哦对了，还有摄像头。乐凡F2配备了一颗后置500万像素摄像头外加补光灯，位于背面左上方。虽然个大，偶尔的拍摄乐凡F2还是能轻松应对的。


后置500万像素摄像头

　　外观样式就是这样，在与富士康合作后，乐凡F2在整体工艺上没有太大的瑕疵。非要说可以提升的地方的话，笔者觉得可能就是机身背面的材质了，笔者是比较喜欢高端霸气上档次的金属机身的，当然，材质问题是一个仁者见仁的问题。

酷睿i5芯保驾护航

　　如今有不少机器（PC或者平板或者变形产品）都成为了Windows 8的载体，其中有很多都搭载了Intel Atom Z系列处理器，说到Atom消费者难免会联想到曾经昙花一现的上网本，或者说，搭载Atom处理器的平板就像是上网本变了个样儿再与消费者见面一样。从之前看到的大伙评论中，搭载更高规格处理器的平板产品似乎更受欢迎。

　　我们拿到的乐凡F2是F2系列钻石版的一款，搭载Intel Core i5-3339Y处理器，标配4GB内存和64GB存储空间，支持WCDMA 3G网络，价格为4999元。


Intel Core i5-3339Y处理器

　　钻石版搭载的处理器都为Intel Core i5-3339Y，极速版使用的是Intel赛扬1037U处理器。这颗Intel Core i5-3339Y信息如下，22nm工艺，1.5GHz主频，睿频后可达2GHz，TDP为13W，融合HD 4000显示核心。

　　Intel尾数为Y的处理器为超低电压处理器，尾数为M为标准电压，尾数为U为低电压。Y处理器的出现是为了解决Atom性能不足以及U处理器功耗较高的问题，它算是一个比较折中的解决方案。

　　这台乐凡F2拥有64GB存储空间，除过系统占用后可用空间大约为44GB，基本的应用需求可以满足，再小点的32GB使用起来容量可能会捉襟见肘，更高容量的64GB/128GB肯定会富裕一些，想要用的舒心自然要拿人民币来换，64GB与128GB之间的差价为600元。


64GB硬盘可用44GB

　　FHD分辨率算是目前的高配水准，不过说实话，在Windows传统界面下，100%大小时字还是比较小的，长时间看屏幕的话眼镜容易疲劳，不过在“豆腐块”Windows UI下效果就非常细腻。

基本信息

　　在大家熟悉的系统评分中，乐凡F2的瓶颈分为图形性能4.3分。从其产品定位来讲瓶颈出现在这里是很容易理解的，毕竟F2的主要使用情况是是轻商务办公，大型游戏可不是它的菜，相信爱玩游戏的朋友直接会选择游戏本或者台式机。


体验指数

　　笔者简单测试了一下乐凡F2的固态硬盘，在持续读写方面这块固态硬盘读取速度为188.4MB/s，写入速度为74.5MB/s，表现中规中矩，4K随机读取速度比较出色。

续航散热总结

　　作为一款全功能电脑，大家熟悉的PS、Office办公等操作在乐凡F2上来说都是小菜一碟，像下图这样，同事就在用乐凡F2完成简单的PPT制作。


office操作无压力

　　笔者在测试续航时使用该机连续播放1080p规格高清视频，乐凡F2可以流畅播放，影音能力表现不俗，通过HDMI线输出到大屏幕欣赏时效果更上一层楼。在连续播放高清视频2个小时后，乐凡F2掉电51%，由此估算在这种模式下该机的续航时间为4个小时左右。如果是穿插上网、办公应用的话相信续航时间能更长一些。


1080p视频播放无压力

　　在视频播放大概30分钟时，笔者使用热成像仪扫描了乐凡F2的发热情况，具体如下图所示。出风口温度最高，接近47°，风扇也在高速运转，中部靠上位置温度在38°左右，双手握持的边角部位比较温度比较温和。总体来看，乐凡F2的温控表现还是可以接受的。

PConline评测室总结

　　移动互联这块大蛋糕人人都想吃一口，微软携Windows 8强势来袭，本来就竞争激烈的平板市场这下更热闹了。经过大半年的发展，Windows 8与iOS、Android三足鼎立之势正在慢慢形成，更方便内容创造是Windows 8相对于后二者的最大优势。
