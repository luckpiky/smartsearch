Tegra 4加触控笔 Tegra Note 7平板评测
手机中国  2013-11-19
原文地址：http://pad.cnmo.com/31/313334.html

　　【手机中国 平板】我们知道四核平板早在2011年就已经出现，在那个时候，四核芯对于一款平板来说是一个天大的事情，给人的感觉就是性能十分强悍。而作为生产全球首款四核芯处理器的厂商：英伟达，在2013年又给了我们一次惊喜，虽然这次还是四核，但相比上代的四核要强悍太多。

　　这次英伟达和汇威合作推出了搭载英伟达Tegra 4处理器的7英寸平板：Tegra Note 7，这款平板不仅非常便携，同时自带的手写功能也是非常独特。

　　究竟这款全新四核平板会有什么样的表现呢？接下来笔者就通过一系列的评测为大家揭开这款平板的神秘面纱，请大家接着往下看。

机身黝黑精深内敛

　　在外观大小上，采用7英寸设计，基本上是nexus 7的大小一致。7英寸在目前的平板尺寸当中算是比较小的一款，虽无法提供大尺寸平板带来的强视觉效果，但是便携优势尽显。屏幕像素是1280×800像素。

　
Tegra Note 7外观

　　正面的顶部和底部都设置有扬声器，这也难免让人在外观设计上联想到了早前的HTC one的设计风格，上下的两个扬声器的设计可以更加有效地保证了音效的环绕。、前置30W摄像头，这里略显不足。

　
Tegra Note 7细节

　　背面是采用复合材质制作，细密的纹理设计，增加了手与平板之间的摩擦，同样让手在握住平板的过程中有相当不错的手感体验。


Tegra Note 7触控笔

　　针对外观细节方面，我们再进一步看。在平板的旁侧有一个笔槽，用于装备手写触控笔。

原生系统简单优化

　　英伟达Tegra Note 7平板平板电脑运行主流的Android 4.2.2操作系统，并在原生Android基础上简单优化，操作界面区跟原生系统几乎一模一样，英伟达Tegra Note 7取消了解锁界面，按电源键直接进入主界面。

　
英伟达Tegra Note 7操作界面

　　英伟达Tegra Note 7平板下拉是一个特色，原生Android快捷按钮只能打开或关闭，Tegra Note 7的快捷按钮轻按进入设置，长按打开或关闭，实用性很强，刚上手是会有些不适应，习惯了就会感觉很方便。


英伟达Tegra Note 7操作界面

　　英伟达Tegra Note 7可以实用谷歌商店，大家也知道在国内上市的平板绝大多数都不能使用谷歌商店，而且国内应用市场都不待见平板电脑，而谷歌商店有有平板电脑频道，下载安装应用更加方便，英伟达Tegra Note 7有专为Tegra 4处理器打造的游戏市场，下载游戏不用担心兼容性问题。

英伟达恋上触控笔

　　英伟达Tegra Note 7拥有一支功能强大的触控笔，插在机身背部，笔尖可以更换，拔出触控笔系统自动弹出手写应用，此时屏幕底部的3个虚拟按键变成5个，点击最左边的小手进入触控笔模式，在触控笔模式下只能用手写笔操作，再次点击关闭该模式。


英伟达Tegra Note 7触控笔

　
英伟达Tegra Note 7触控笔

　　最右边是截屏编写键，用触控笔点击后自动截屏，可以在屏幕上随便涂鸦，用触控笔随意画圈可选中，圆圈内部分便成为写字板，可随意书写、涂鸦并保存。截屏编写也有不足，没有橡皮功能，每次写错不得不全部取消重新来过。

英伟达Tegra Note 7触控应用

　　英伟达Tegra Note 7内置多款手写应用，拔出触控笔时自动弹出，功能比较全面，实用性也较强。


英伟达Tegra Note 7触控应用

Terga 4性能超级棒

　　英伟达Tegra Note 7采用性能强悍的Tegra 4处理器，是目前性能最强悍的移动终端处理芯片之一，其采用“4+1”的设计，具有四颗主核芯和一颗省电核芯，基于ARM的Cortex-A15架构，具备72个定制NVIDIA GeForce GPU光栅化处理单元，其GPU处理性能是Tegra 3的六倍，网页浏览能力提升2.6倍。

　
安兔兔

　　英伟达推出Tegra Note 7平板电脑就是为了推广Tegra 4处理器，并不是靠平板赚取利润，因此这款平板的价格十分诱人。超强Tegra 4处理器、1GB的运行内存、16GB的存储空间，经笔者实际测试，在性能评测软件安兔兔上的跑分为33771分，属于顶尖水平。

性能测试

　　就下来笔者使用多款测试软件对英伟达Tegra Note 7平板进行全面的测试，通过结果我们可以看到，这款平板的成绩属于顶尖水平，对付日常的应用以及大型的游戏绝对绰绰有余。Tegra Note 7的处理器性能还可以调节，性能与功耗兼顾。

发烧游戏轻松搞定

　　英伟达Tegra Note 7采用性能强劲的Tegra 4四核处理器，主频高达1.9GHz，搭配1GB的运行内存，16GB的存储空间，支持10点触控，性能方面完全不用担心，目前还没有Tegra 4运行不了的游戏。


发烧游戏测试

　　笔者试玩了多款不同类型的游戏，都能完美运行，没有丝毫卡顿，加载速度也非常快，暂时还没有发现这款处理器应付不了的游戏（兼容性问题除外），所以说这款平板性能至少在2年内不会落伍。

1080p视频不在话下

　　在视频播放方面，英伟达Tegra Note 7采用性能超强的Tegra 4四核处理器，解码能力自然不用担心，支持1080P全高清视频播放，支持MKV、3GP、MP4、TS、FLV等绝大多数主流视频格式的解码，还支持超高清4K视频解码。


1080p视频播放

　　在测试中几种主流格式的高清视频都能够正常播放，没有出现卡顿的现象，全高清屏画面显示十分清晰细腻，而且双扬声器的设计，音效也很不错，有立体声效果。


1080p视频播放

　　不过，英伟达Tegra Note 7也有不支持的视频格式，如果想支持所有的视频格式，需要下载第三方解码软件，提醒大家注意视频播放软件虽然能播放全格式的视频，但功耗相对较大。

拍照功能十分强悍

　　英伟达Tegra Note 7主要有三个优势，首先是采用性能超强的Tegra 4处理器，其次拥有功能全面的触控笔，最后是强大的拍照功能。


拍照界面

　　英伟达Tegra Note 7的拍照界面十分华丽，右边是拍照键，轻按右上部的白色三角弹出设置，选项特别全，凡是单反可以调节的东西它都有，还可以添加黄金分割、三角分割等四种辅助线，最重要的是可以曝光锁定，这个功能秒杀其他平板电脑。


Tegra Note 7样张

　　这款平板采用的是500万像素的主摄像头和120万像素的前置摄像头，主摄像头最大可拍摄2592×1944像素的照片，支持1080p视频拍摄，基本满足日常拍照需求，在逆光条件下也有不错的拍照效果，以下是英伟达Tegra Note 7的拍照样张。


Tegra Note 7样张

　　英伟达Tegra Note 7摄像头像素虽然不是很高，但在表现广阔场景的时候依然很给力，画面层次分明，远处的景物依稀可辨，色彩还原真实，放大1倍也没有明显噪点。


Tegra Note 7样张

　　曝光锁定是单反相机上的功能，目前没有一款平板拥有该功能，英伟达Tegra Note 7除外，下面笔者简单介绍一下用法，在拍照界面任意地方点击自动对焦，自动对焦的方框里有一个圆圈是曝光点，长按圆圈则锁定曝光值，此时任意场景下都采用该曝光值，再次点击圆圈关闭曝光锁定，利用这个功能可以拍出十分有趣的照片。

总结：

　　英伟达推出Tegra Note 7并不是为了靠平板赚取利润，而是起指导性作用，方便下游平板厂商推出更好的平板电脑，作用与谷歌Nexus 7相同（Nexus7是Android平板标杆），所有Tegra Note 7平板的价格十分诱人。虽然这款平板的外形谈不上时尚，但其有性能强劲的Tegra 4处理器，实用的触控笔，还有强大的拍照功能，堪称谷歌Nexus 7二代最强对手。
