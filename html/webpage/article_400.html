谁让你心动?2013大尺寸平板年度盘点
中关村在线  2013-12-22
原文地址：http://pad.zol.com.cn/418/4187592.html

    2012年末问世的iPad mini可以说引发了一场平板行业的革命，这款拥有7.85英寸机身的小尺寸平板因为体型小巧便于携带，深受消费者的欢迎与喜爱。而无论是iPad还是其它传统大尺寸平板产品，无不因为它的问世而在销量方面受到了很大的冲击。那么小尺寸平板是否就可以直接替代传统大尺寸平板呢？并非如此。

    虽然小尺寸平板在便携性上有着无与伦比的优势，但由于屏幕尺寸的限制，我们无法在该类产品上获得比较出色的视觉体验。无论是玩游戏还是看电影，毫无疑问还是使用大屏幕平板产品带来效果的更出色一些。所以总的来说，小尺寸平板是一款适合我们外出携带的产品，而如果在家中使用的话还是传统的大尺寸平板更实用一些。



    所以回顾2013年我们可以看到，虽然目前所有的平板厂商都在马不停蹄推出自家的小尺寸平板产品，但同样的也没有把大尺寸平板落下，像iPad Air、三星 Galaxy Note 10.1（2014版）都是接近年末问世的优秀产品。同时今年国内的许多平板厂商也开始发力，像联想 YOGA TABLET 10、蓝魔i9等产品，同样是极具特色的优秀产品。


大尺寸平板仍然是市场不可或缺的一部分

   2013年毫无疑问是平板行业呈井喷式爆发的一年，虽然市场上不断的涌现出不同品牌不同形态的平板产品，但要说真的从中挑出各方面都堪称优秀的产品，那绝对是凤毛麟角。所以本文笔者还是主要从一些知名品牌下手，为大家带来本年度大尺寸平板的年末盘点。

    本文将为大家详细介绍12款不同品牌的大尺寸平板产品，这些平板无论是工业设计、硬件配置、起售价格还是功能等方面都不相同。希望大家可以通过本篇文章对这些平板产品有一个全新的认知，从而挑选出一款最适合自己的平板产品。

    联想作为国内数一数二的数码品牌，在自家的平板产品身上一直赋予了许多与众不同的创意。比如去年为大家带来的YOGA 11有四种使用模式，而今天笔者为大家带来的YOGA TABLET 10主打三种使用模式。看似少了一种，但细看产品形态就会发现，YOGA 11是笔记本设计，YOGA TABLET是平板设计，两者不能混为一谈。

★外观设计：



YOGA TABLET 10机身正面


YOGA TABLET 10机身背面

    YOGA TABLET 10采用了书卷式的设计理念，在两年前索尼曾经推出过这类产品，前重心偏置的确让人眼前一亮。但YOGA TABLET 10这个设计可以说是索尼书卷设计的升级版本，将电池集中于支架上，既可以保证机身的轻薄同时也可以将大容量电池装置在圆通支架中。

★特色亮点：


联想YOGA TABLET 10毫无疑问具有很大的创新意义

 
三种不同使用模式

    依靠这个巧妙的书卷支架设计，YOGA TABLET 10功可以为我们提供三种使用模式——阅读模式、站立模式以及触控模式。阅读模式则是在于书卷式设计带来的良好吃我手感，站立模式则是可以依靠支架功能让机身稳固站立在桌面上，让我们可以以最舒适的姿态观看屏幕。触控模式其实是站立模式的另一版本，它分为两个角度，在不展开支架的状态下凭借圆柱形转轴提供阅读角度；打开支架后，角度进一步增加，这个状态下的造型十分有个性，相信能吸引不少眼球，独一无二。

★硬件配置：


    硬件配置方面，YOGA TABLET 10配备了10.1英寸屏幕（支持10点触控），分辨率为1280*800，搭载了MTK四核MT8125处理器，主频为1.2GHz。可以说这个配置在目前千元左右的平板产品中十分的可见，这也从侧面告诉了我们联想YOGA TABLET 10不是一款主打硬件性能的平板产品。

★笔者观点：

    对于一些“紧抓”硬件配置不放的朋友来说，YOGA TABLET 10肯定不值它那2200元的售价，但自iPad发布以来，真的还没有一款像YOGA TABLET 10这样的平板产品来打破传统平板的形态。此外，作为一款国际大厂的平板产品可以说无论是工艺设计还是系统优化都要甩开同配置的国产平板一大截，售后服务方面那更是没有任何的可比性。花2200元购买该款平板产品虽然并不算一件十分超值的事情，但也绝对的物有所值。

    三星近些年可以说为我们带来许多出色的智能设备，该品牌每每推出新品往往都会引起消费者的强烈关注。为了继续挑战苹果iPad在平板行业的霸主地位，三星在今年10月份适时推出了一款可以说是在各个方面都堪称顶级的平板产品——Galaxy Note 10.1（2014版）。

★外观设计：



Galaxy Note 10.1（2014版）机身正面


Galaxy Note 10.1（2014版）机身背面

    相比上代产品，Galaxy Note 10.1（2014版）最大的变化就要数机身变得更加轻薄便携。除了将设备的厚度降低到7.9毫米之外，Galaxy Note 10.1（2014版）前面板屏幕两侧的边框，以及原来在边框边缘处的立体声扬声器相比前作都有一定程度的缩减。总的来说Galaxy Note 10.1（2014版）虽说依然秉承了三星一贯的设计风格，但却用另外一种方式让工业设计变得更加完美。

★特色亮点：


三星Note 10.1（2014版）配备的S Pen的确可以为我们创造更多内容

    要说到该款产品最大的特色，那莫过于它随身携带的那根S Pen触控笔了。依靠S Note应用可以记录绘制出精美的备忘录、笔记，并且这些记录还可以被导出成PDF文件或者是图片进行分享传播。此外通过S Pen我们还可以呼出环形悬浮指令菜单，进行一些应用的快捷操作。当然对于这样一款“高富帅”产品来说，它的亮点绝对不仅仅只有这些，像分屏多任务、眼球追踪等功能同样是该款产品独有的。

★硬件配置：


    Galaxy Note 10.1（2014版）配备了一块2560x1600超高分辨率的LCD屏幕，同时还搭载了三星Exynos 5420四核1.9GHz+四核1.3GHz处理器。可以说单凭硬件配置来看，市面上似乎还没有一款产品能与Galaxy Note 10.1（2014版）相匹敌。

★笔者观点：

    优秀的工业设计、顶级的硬件配置、出色的系统体验，一款优秀平板的这三大要素可以说Galaxy Note 10.1（2014版）全都具备。虽然该款产品售价高达3788元，但在这个价位中除了iPad Air之外毫无疑问Galaxy Note 10.1（2014版）是我们的最佳选择。

    蓝魔虽然还没有像联想等国产厂商打入国际市场，但在国内平板市场中绝对算的是响当当的品牌。这一部分是依靠前几年生产MP3/4打下的基础，但绝大部分还是因为蓝魔真的是在用心做产品。尤其是在牵手英特尔后推出的这款i9平板电脑，更能看出蓝魔要与国际品牌掰手腕的雄心壮志。

★外观设计：



蓝魔i9机身正面


蓝魔i9机身背面

    蓝魔i9做工工艺相当的独特，抛光的CNC一体化机身工艺让其显得档次一下子不同于很多国产平板，有了不少的国际范儿。白色面板透露着十分简约的设计理念，前面板中除了前置摄像头之外没有多余累赘，整体十分的简洁大方。虽然外观设计方面仍然感觉有模仿苹果的痕迹，但其极简清新风格相信仍然会赢得不少消费者的喜爱。

★特色亮点：


AAC独立双层音腔（图片来自蓝魔官网）

    作为一款售价仅为千元的产品，我们很难在蓝魔i9身上看到太多的与众不同的地方，毕竟它在工业设计以及硬件配备方面已经给我们带来了足够的惊喜。所以该款产品所拥有的AAC独立双音腔可以说是为数不多的一个亮点，该项设计可以使设备拥有更大的外放音量，使得声音饱满清晰，同时能够带来立体声的环绕效果，让用户能够在适当的情况下抛弃耳机，从而拥有更好的视听享受。

★硬件配置：


    前文提到，蓝魔i9配备了一颗英特尔Atom架构的Z2580双核四线程处理器，主频为2GHz。经过安兔兔检测，这颗处理器性能竟然高达三万分。屏幕方面蓝魔i9则拥有一块分辨率为1920x1200的分辨率，这在目前的平板产品中也属于较高的规格。

★笔者观点：

    无论是外观设计，模具做工，功能配置以及硬件性能方面来看，蓝魔i9都称得上尽善尽美，让人很难挑出明显缺陷。在用户体验方面，由于英特尔强芯助力使其在Full HD级显示分辨率下依然有堪比大牌的流畅体验。可以说蓝魔i9完全颠覆了我们对国产平板以往的形象，做到了真正的低价也有好产品。

    华硕这两年在平板行业中的发展想必大家一定有目共睹，无论是自家的旗舰产品还是为谷歌代工的Nexus系列，在市场上都获得了不错的反响。而这款代号为TF701T的大尺寸平板产品，则主打“二合一”的理念。

★外观设计：



华硕TF701机身正面


华硕TF701机身背面

    如果不加以产品线细分，华硕Transformer Pad (TF701T)摆放在那里更像是一台小尺寸的笔记本，金属顶盖、IPS屏幕、实体键盘和左右接口布局几乎一应俱全。作为一款二合一平板产品，华硕TF701T采用10.1英寸机身规格，顶盖发丝纹路依然引用了华硕移动产品线家族式的设计元素。整机金属质感十足，顶盖边缘和转角细节做工很是到位，圆滑的手感握持也非常舒适。

★特色亮点：


键盘打字手感还不错

  
键盘左下角的Home键和Fn快捷功能键、一体化触控板设计 

    该款产品最大的亮点毫无疑问是随机附带的这个键盘底座。为了迎合Transformer Pad (TF701T)搭载的Android系统，该款键盘底座的键位也经过了特殊订制，左下角的Home按键和最上一排的快捷功能按键都与Windows键盘有不小的区别，虽然大家使用时需要一段时间记忆和适应，但习惯了以后快捷按键的操作方式还是可以帮助我们化简不少的Android调整步骤。除此之外该款键盘底座还内置电池，让续航更有保障。

★硬件配置：


    从键盘底座就可以看出华硕的这款新品走的是高端路线，所以在硬件配置方面同样不俗。在处理核心方面选择了目前移动处理器中性能最为强劲的英伟达Tegra 4芯片，  代号为蝙蝠侠“Wayne”的Tegra 4处理器具备72个定制的NVIDIA GeForce GPU显示核心，其GPU处理性能是Tegra 3 的六倍。屏幕方面也配备了一款分辨率为2560×1600高清面板，显示效果极为出色。

★笔者观点：

    总体来说，华硕TF701T是一款延续型产品，创新方面略显不足，高分屏和硬件平台是该机主要卖点。当然作为一款售价高达3988元的产品它还具备许多Android平板不具备的特点，如USB3.0接口的配备、SonicMaster技术等都是该款产品的高端之处。

    虽然七彩虹进入平板行业时间并不长，但我们看到它从DIY行业积攒下来的研发能力与超高人气助其平板一路扶摇直上，迅速成为市场中关注度最高的国产平板品牌之一。CT102 Qise 3这款产品是七彩虹在今年下半年推出的一款大尺寸平板产品，可以说该款产品是一款极具“国产特色”的平板产品。

★外观设计：



CT102 qise 3机身正面


CT102 qise 3机身背面

    CT102 qise 3采用了国产平板常见的三层复合式结构，机身接缝都非常紧密平整。整机虽没有追求当下流行的轻薄式设计，但黑与白的配色以及刚柔并济的外形线条时尚感十足，是一个看到第一眼就会让人爱上的漂亮平板。

★硬件配置：


    除了外观之外，硬件配置方面CT102 qise 3也选择了目前许多国产平板都在选用的全志A31S芯片解决方案。该款芯片组采用的是四核心Cortex-A7架构通用处理核心，主要的特点便是低功耗。屏幕方面可以说CT102 qise 3配备的相当不错，这块分辨率为1920×1200的屏幕可以说已经达到了视网膜级别，画面质感极为细腻。

★笔者观点：

    作为一款标准的国产平板，要想在CT102 qise 3找到与众不同之处是在有些强人所难。不过可以说作为一款售价千元的产品，CT102 qise 3绝对可以满足用户日常使用的所有需求。此外七彩虹作为国内知名的企业在产品的质量以及售后方面也会给用户十分不错的保障。

    台电可以说也是国内比较知名平板厂商之一，该品牌的发展道路同蓝魔等国产厂商几乎一样，都是从当年生产MP3/4慢慢转型生产平板电脑。这款本月初刚刚问世的P11HD大尺寸平板产品，可以说是台电在2013年的封山之作。

★外观设计：



台电P11HD机身正面


台电P11HD机身背面

    台电P11HD可以说在外观方面有了很大的突破，采用全新的外观模具使得该款产品的机身厚度极为轻薄，仅为8.5mm。机身边框也采用了全新的超窄设计，达到了11.5mm。可以说在目前的大尺寸平板当中，台电P11HD虽然称不上最轻薄的那一款，但也绝对在前10名之中。

★特色亮点：


eMMC存储技术的确让产品在性能上更有保证

    国产平板虽然由于各方面因素没有什么的打的创新，但偶尔也能发现一两个小亮点。比如台电P11HD就使用了国际上最为先进的eMMC存储技术，相比较其它国产平板在稳定上和运行速度上都有一定的提升。

★硬件配置：


    在硬件配置方面台电P11HD只能算是中规中矩，陪伴了RK3188四核处理器，主频为1.6GHz。屏幕方面该款产品倒是选择了不错的配置，1920×1200分辨率的屏幕显示效果足够令人满意。

★笔者观点：

    为了保持在价格方面有足够的竞争力，国产平板始终不能在工业设计和硬件配置都让用户感到满意。台电P11HD最大的卖点在于机身十分的轻薄，但性能表现十分的一般。由于该款产品目前还在预售阶段，所以目前还不能知道最终的售价。如果最终在千元左右的话，相信大家还是可以接受的。

    五元素也是大家比较熟知的国产平板厂商，可以说这一年下来该品牌也为我们带来了许多不同类型的产品。该品牌给笔者最大的印象除了印在后壳处那个大大的数字Logo之外，便是在产品外观设计上总喜欢与众不同。而这款ifive X3，相信绝对颠覆了许多朋友对国产平板的看法。

★外观设计：



五元素ifive X3机身正面


五元素ifive X3机身背面

    首先五元素ifive X3在配色方面就十分的夺人眼球，黑加藏蓝的配色在平板产品中绝对罕见。此外便是该款产品的轻薄程度绝对足够的出人意料，经测量整机厚度为6.98mm，重量则为548g，“最轻薄的国产大尺寸平板”的名号绝对当之无愧。

★特色亮点：


ifive X3系统主界面


ifive X3二级应用菜单

    ifive X3除了轻薄性之外拥有独一无二的ifive-skin系统也是一大亮点，虽然该系统比起MIUI、甚至是锤子系统的开发工作量都弱很多，但相对于其它国产平板前篇一律的原生系统，ifive X3绝对能给消费者带来耳目一新的感觉。

★硬件配置：


    ifive X3同样采用了RK3188四核处理器，与国内其它平板产品并没有拉开实质性的距离。但在其它方面我们绝对可以看到无元素满满的诚意，把2GB机身内存、32GB机身容量以及1920×1200高清屏幕作为标配的国产平板笔者还真是从未见到。

★笔者观点：

    总结来说，能够将10.1英寸平板做到如此轻薄、漂亮、精致而配置全面的国产厂商我想用5根手指就足以数得过来，并且成功的案例也只有ifive X3这一款产品。笔者可以肯定的说，作为一部千元级平板ifive X3已经做的足够出色。

    老牌的国内数码品牌原道相信关注平板电脑产品线的读者们再熟悉不过了，产品更新速度快并且价格优势大，质量方面也有很大的保证。在如今许多国产平板品牌都陆续进入“轻薄时代”后，原道自然也不甘落寞的推出自家主打轻薄的大尺寸平板产品——原道M10。

★外观设计：



原道M10机身正面


原道M10机身背面

    原道M10机身采用CNC切割工艺，模具的切割采用数控机床，与普通机床相比加工精度更高质量更加稳定。机身正面边框较窄，前置200万像素摄像头位于左上方的位置，其余没有多余的元素进行装点。虽然官方宣称该款产品的厚度仅为7.1mm，但经过实测之后则为7.56mm，与iPad Air相比基本保持一致的水品，可以说也是一款十分轻薄的平板产品。

★硬件配置：


    配置方面原道M10与前面介绍的几款国产平板无太大变化，均为RK3188四核处理器外加1920x1200高分屏的配备。可以说现如今国产平板同质化严重的状况依然没有得到很好的解决，而厂商们似乎也十分的享受目前大家共享一杯羹的局面。

★笔者观点：

    随着国产平板在工艺上的不断提高，原道M10在外观方面的确的十分惹人喜爱。但不得不说的是，拥有相同配置同时售价在千元左右的平板产品着实有很多，相比之下原道M10虽然依然极具竞争力，但也没有太过突出的优势。

    品铂也是大家比较熟知的一个国产品牌，虽然踏入平板行业也不算太久，但作为从代工厂转型而来的品牌，品铂在为别人做嫁衣的同时也积累了上游渠道资源与产品制造经验，推出自有品牌产品时自然就有了先天优势。

★外观设计：



M9 Pro机身正面


M9 Pro机身背面

    凭借代工厂出身的优势，品铂平板几乎从不会与其他品牌的机型“撞衫”，所以这款M9 Pro所拥有的独立外观设计在同质化严重的国产平板中显得尤为难得，灰色的机身配色倒是很符合品铂大尺寸平板一贯的特点，庄重而大气。尤其是在机身背面使用了具有拉丝金属工艺制成的面板，对机身整体的档次有很高的提升。

★特色亮点：


品铂独有的UI界面


分屏显示效果

    品铂M9 Pro与其它国产平板与众不同之处便在于对Android系统进行了一定的优化，类似于Win8风格的界面可以让用户能够一目了然的区分应用的位置。除此之外，品铂M9 Pro还支持分屏功能，在桌面上我们可以同时打开多个窗口进行操作。

★硬件配置：


    品铂与国内芯片厂商瑞芯微一直都保持着十分良好的关系，所以品铂M9 Pro依然采用了RK3188处理器。作为前作M9的升级版本，M9 Pro除了核心处理方案没变之外在其它方便都有了不小的提升，1080P级别屏幕的配备以及3G模块的融入的确让整款产品提升了一个档次。

★笔者观点：

    品铂M9 Pro可以说是比较全面的一款国产平板，几乎一款平板该具有的功能它全部具备。此外3G版本与WiFi版本的售价也只不过相差了200元，价格方面也相当的厚道。

    普耐尔这个品牌虽然成立的时间十分早，但与其它国产品牌相比似乎稍显得有些不温不火。虽说如此，该品牌为自家平板产品起的名字倒是十分的朗朗上口，“MOMO”这个名称相信有不少消费者都已经铭记在心。此外作为一个国产老牌厂商，普耐尔生产的平板产品在质量上还是很有保障的。

★外观设计：



 MOMO 20HD机身正面


 MOMO 20HD机身背面

    MOMO 20HD是今年上半年问世的一款平板产品，虽然外观方面并没有摆脱苹果所定下来的固有模式，但整体看上去还是十分的简洁大方。其背面采用的是复合材料配合金属喷砂工艺，整体金属质感很强烈。虽然是复合材质，但是用力按压强度还是非常高的，对整个机身的保护效果不俗。

★硬件配置：


    普耐尔MOMO 20HD采用了全志A31S四核处理器，主频为1.2GHz，配备了PowerVR系列GPU，在游戏方面还是能为我们提供一定的保障。作为一款“HD”级别产品，普耐尔MOMO 20HD自然拥有一块显示效果不错的屏幕，分辨率为1920×1200。

★笔者观点：

    国产平板同质化严重的情况再一次在这款普耐尔MOMO 20HD身上得到了印证，无论是千元的售价还是硬件配置都与许多其它品牌的产品保持一致。作为一个成立近10年的数码品牌，普耐尔应该花更多的心思来让自己的产品变得与众不同一些。

    索尼作为日本最具有代表性的科技品牌，无论是在任何领域似乎都给我们带来过不少的惊喜。虽然在该品牌推出的平板产品并不算多，但不得不说真的是个个都属精品。今年上半年问世的这款Xperia Tablet Z平板电脑，可以说绝对是日系厂商的代表之作。

★外观设计：



索尼Xperia Tablet Z机身正面


索尼Xperia Tablet Z机身背面

    虽然索尼Xperia Tablet Z没有采用Unibody一体成型机身制造工艺，但是方方正正的机身造型与机身边缘的巧妙贴合设计让该机看上去整体感很强，就好像是在一整块晶体上切割而成的，没有其它同类产品上那种非常明显的拼接痕迹，另外正面的屏幕也与机身的边框很好的融合在了一起，锁定屏幕时我们很难用肉眼区分显示屏与边框的界线。

    此外不得不说一下该款产品的轻薄特性，作为目前世界上最轻薄的大尺寸平板的产品，索尼Xperia Tablet Z的机身厚度真的为6.9mm，而482g的机身重量更是无人能及。我们可以看到目前许多平板厂商在年末推出的产品都是主打轻薄特色，但至今还没有一款产品可以打破Xperia Tablet Z的记录。

★特色亮点：

 


市面上的三防平板真的不多见

    日系厂商似乎特别钟爱推出三防产品，所以我们可以看到索尼Xperia Tablet Z也拥有着三防功能——防尘、防水以及防震。在官方给出的数据显示，这款平板的防护级别达到了IP55/IP57的水平，可以有效的防止有害粉尘的堆积，同时用水冲洗机身以及短时间的浸泡对该机都不会产生伤害。

★硬件配置：


    作为一款上半年问世的产品，索尼Xperia Tablet Z并没有配备目前最新问世的处理芯片。但其采用的高通骁龙APQ8064与2GB RAM的组合在目前也绝对属于十分顶级的配备，其性能不是售价千元的国产平板所能相比。1920*1200分辨率的屏幕虽然与目前售价三千多元的产品还有着轻微的差距，但是在索尼 Mobile Bravia Engine 2显示引擎的渲染下，这块屏幕的显示效果还是可圈可点的。

★笔者观点：

    索尼 Xperia Tablet Z的出现为整个平板市场带来了的新的活力。在笨重的新iPad面前，它就好像是放置在笔记本电脑面前的超极本，6.9mm的机身厚度以及485g的机身重量都超出了我们对平板电脑的传统印象，原来大尺寸平板还可以做的如此轻薄。另外没有什么明显的短板是这款产品最大的优势所在，它也没有因为体积的缩减造成功能上的缺失或者是体验上的下降。目前该款产品售价约为3200元左右，依然有着对抗iPad Air实力。

●写在最后：

    回顾2013年，我们可以看到大尺寸平板产品并没有随着小尺寸平板的盛行而衰落，反而依然具有联想YOGA TABLET 10、索尼Xperia Z等具有创新意义的产品出现。除此之外，芯片大厂英特尔也开始携带X86架构的移动处理器进军Android市场，这无疑更为整个平板市场带来了新鲜的活力。

    此次笔者为大家带来的这些大尺寸平板产品采用的均是Android系统，对于绝大部分用户来说购买该类平板普遍都会用于看电影、玩游戏等娱乐项目。虽然Win8平台的游戏应用数量也在不断提升，当相信在短时间内还无法拉近与Android平台的差距。相信在未来的2014年，仍然会有更加出色、更具有创新意义的大尺寸Android平板与大家见面。
