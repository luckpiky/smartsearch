手写全能旗舰平板 三星新Note10.1评测
手机中国  2013-11-10
原文地址：http://pad.cnmo.com/30/306856.html

　　【手机中国 平板】今年绝对是平板电脑井喷的一年，平板电脑被重新发明进入到了第三个年头，其发展模式已经和智能手机相似，虽然在出货量上还无法赶超手机，但其销售数量已经快要超越传统PC，可以说平板电脑不仅让我们的生活更加的便利，同时也成为智能手机除外人们最可或缺的数码产品。

　　平板电脑如今被两大阵营牢牢占据，以苹果为主的iPad系列平板占据了大多数，但这个数据已经慢慢被Android阵营赶超，毕竟一个品牌无法与如此多的Android系统OEM厂商抗衡，而在Android阵营中最为出彩的代表就要属三星，因为其产品系列非常之多，同时产品线也是最广的，覆盖了高中低各种档次。

　　这其中要数“Note”系列产品最为成功，虽然在大尺寸的平板上的产品仅仅有三款产品，但作为三星平板中的旗舰产品，这一代的GLALXY Note 10.1自然不会让人们失望，虽然三星莫名的加入了2014版这个称为，但如今我们在2013年拿到这款产品还是可以看到2014年一些新产品的影子，究竟这款平板相比上代有什么样的提升，接下来的这篇评测会为大家解读。

延续今年整体设计理念

　　GALAXY Note 3与这款平板是一同问世的，这两款产品的整体设计延续了三星今年整体的设计，但一些细节还是有略微的差别。GALAXY Note 10.1 2014版同样采用了触感柔软、仿皮革的材料，三星真的已经把塑料材质运用到炉火纯青，即便是看似皮革，手感也相同的后盖其实也是塑料。这样带来的好处就是不易沾染指纹，而且手感的阻力也更大，握持的感觉更好。


GALAXY Note 10.1 2014版正面

　　与iPad Air一样，今年大尺寸的平板在整体的尺寸上都选择了缩减，同时也引入了今年智能手机上流行的窄边框，这让设备整体看起来更加紧凑，科技感更为强烈。


GALAXY Note 10.1 2014版背面

　　上代产品边框在的镀边过为显眼，而在这代产品上，边框全部放在的机身的四周，让使用者把注意力更多的放在了屏幕上而不被边框所吸引。机身整体没有了上代鹅卵石版的圆润造型而是换做了棱角更为分明的框架，使得整体凸显的更为简洁，手持效果也比上代轻松不少。


GALAXY Note 10.1 2014版按键细节

　　按键和旁边的两颗虚拟按键上这代产品的设计一致，但和手机不同的是虚拟按键不需要背光灯就可见。但在GALAXY Note 3就不是这样。


GALAXY Note 10.1 2014版前置摄像头细节

　　前置摄像头和距离传感器在屏幕顶部偏左的位置，没有GALAXY Note 3上的LED指示灯。


GALAXY Note 10.1 2014版背部纹理

机身细节展示

　　GALAXY Note 10.1 2014版改变了边框的喷漆技术，同时还增加了数条齿状花纹，增强边框的摩擦力使得手感上更为出色，同时使得整款平板的质感更为出色。


机身边框的卡槽

　　机身旁边的SIM卡槽和Micro-SD卡槽的设计与边框纹理设计的比较一致，没有打破边框的纹理设计，统一性较好。


手写笔特写

　　而在机身的右侧则是放手写笔的位置。和上代手写笔在底部右方的设计不同，这次的设计更为方便的取出。


音量按键特写

　　实体按键和上代设计也是有很大的不同，同时在按键的手感上相比上代更为出色。


机身边框的设计纹理

　　机身四周的包边和细节方面做的相当出色，三星在智能手机上出色的做工延续到了平板上，这也是为数不多做工精致的Android平板。

与Galaxy Note 3对比

　　前面提到GALAXY Note 10.1 2014版在外观和造型方面延续了GALAXY Note 3的设计，下面我们就针对这两款同时发布的产品进行一个对比。

GALAXY Note 10.1 2014版对比GALAXY Note 3

　　我们看到在很多细节上两款机器的设计都是相同的，包括正面按键的大小以及包边设计都是相同的。


GALAXY Note 10.1 2014版对比GALAXY Note 3

　　背部的皮革版纹理也是不尽相同，但GALAXY Note 3后的Logo有着金属的覆盖，而在GALAXY Note 10.1 2014版上则没有。


GALAXY Note 10.1 2014版对比GALAXY Note 3

　　摄像头参数上两款产品也是不同，GALAXY Note 3采用了1300万像素的镜头，而GALAXY Note 10.1 2014版则是采用了800万像素的镜头，虽然差距不小，但对于一款平板来说800万加闪光灯的配合还是十分强悍的。

　　两代产品在Home键的设计上是一致的，但不同的是两侧的触控按键，GALAXY Note 3是需要点亮屏幕才会有背光显示，而GALAXY Note 10.1 2014版则无需点亮屏幕就可以看到。

　　机身的细节方面两款机器在整体的设计上都很一致，不过GALAXY Note 3有着USB 3.0的接口在传输速度上还是要快于GALAXY Note 10.1 2014版的。

手写笔全面升级

　　在对比完外观轮廓之外大家可不要忘了“Note”系列上最为重要的手写笔，因为这才是这款产品最为核心的理念，如果这杆笔用的不舒服的话则让整机的体验大打折扣。


手写笔对比

　　GALAXY Note 10.1 2014版提升了手写笔的大小，同时相比上代三角体的设计，这代产品扁平话的四角设计更为舒适。


两款手写笔全面对比

　　三星GALAXY Note 3的手写笔整体上要小于GALAXY Note 10.1 2014版的，但笔尖的舒适程度还是相同的。

高清屏幕效果展示

　　三星GALAXY Note 10.1 2014版配备了一块精美的WQXGA（2560×1600像素）TFT LCD面板，达到了299ppi，比iPad Air的264ppi还要高。分辨率是去年机型的一倍。其显示效果可以说是目前Android平板中最为优秀的，但和色准出色的iPad Air相比还是有些差距，饱和度方面则是控制的不错的，相比GALAXY Note 3的AMOLED屏幕要好很多。


屏幕测试效果

　　有着如此高分辨率的屏幕，GALAXY Note 10.1 2014版在显示效果上极为出色，颜色的还原十分准确，绝对是目前Android平板中的一面旗帜。

屏幕测试效果

　　GALAXY Note 10.1 2014版的屏幕效果总的来说还是十分出色的，虽然在亮度上没有达到最高，但在颜色还原和显示效果上则是要好于自家的GALAXY Note 3的。

S-Pen手写功能展示

　　S Pen一直都是三星Note产品最为突出的特色功能，GALAXY Note 10.1 2014版的S Pen在功能上与Note 3保持了一致。其中的特色包括悬浮预览（Air View），也就是将S Pen悬浮在文字、相册、日期或其他项目上，来对其中的内容进行预览。


S-Pen手写功能展示

　　通过在屏幕上方悬浮并按住按键，系统便会弹出5个不同的选项：Scrap Booker会将屏幕上的信息记录起来；Action Memo取代了原有的S Note小部件，让你能够快速记录一份备忘或是电话号码；Screen Write对屏幕进行截图。


手写效果

　　手写笔的准确性更为出色，笔尖相比上代产品更为柔软，因此触感效果特别不错。


手写效果


S-Pen手写浮窗功能

　　S-Pen把以前相对比较分散的功能进行了整合，虽然功能依旧比较繁琐，但用户的体验相比上代有了提升，同时拔出笔唤醒的效果也更为出色。

全新设计的UI

　　GALAXY Note 10.1 2014版搭载了最新的Android 4.3操作系统，当然，还包括三星的TouchWiz定制界面。三星自家的应用商店，S语音，三星Link，智能停留/滚动/暂停/翻转，Group Play，S翻译都包含在内，整体的页面相比上代更为简洁高效。

顶级处理器性能超群

　　三星Galaxy Note 10.1 2014版处理器分为两个版本，LTE版搭载主频为2.3Ghz的骁龙800四核处理器，3G和Wi-Fi版搭载三星自家的1.9GHz双四核Exynos 5420处理器（1.9GHz的A15四核＋1.3 GHz的A7四核)，两款都是现在最顶尖的处理器。


安兔兔跑分测试

　　三星Exynos 5420处理器采用big.LITTLE架构，搭载四颗Cortex-A7核心和四颗Cortex-A15核心，兼顾性能和功耗。在GPU方面，三星使用ARM公司的Mali-T629 MP6图形处理器，并搭配933MHz的双通道32位LPDDR3，最大带宽高达14.9Gbps，图形性能非常强劲。


安兔兔跑分测试

　　笔者拿到的是32GB/3G版，搭载的自然是Exynos 5420处理器，配以3GB的运行内存，32GB的存储空间，性能绝对处于顶尖水平，下面我们就通过几款常用的软件来测试一下Galaxy Note 10.1 2014版的各项性能。


安兔兔硬件检测

　　《安兔兔评测》是一款专门为Android手机、平板电脑进行性能评测软件，也是现在人们最熟悉的评测软件之一，Galaxy Note 10.1 2014版在安兔兔上的跑分高达32734分，仅次于三星自家的Note 3手机（两者采用相同处理器，Note 10.1屏幕分辨率更高，因此拉低了分数），甩其他平板电脑好几条街。


vellamo

　　接下来笔者用Vellamo网页浏览器基准测试软件进行测试，主要包括HTML5和Metal两个模块，HTML5模块用来评估移动网页浏览器性能，而Metal模块可以对移动处理器的CPU子系统性能进行检测。Galaxy Note 10.1两个模块得分为2571和1285分，出于绝对领先地位。


象限

　　最后笔者用Quadrant测试，检测的范围包括12项CPU测试、1项内存测试、4项I/O测试、1项2D图形处理测试、3项3D图形处理测试，分数越高代表被测试设备硬件综合性能越强。Galaxy Note 10.1不负厚望高达7223分。

运行大型3D游戏小菜一碟

　　安兔兔等软件只能从侧面反映处理器的性能，三星Galaxy Note 10.1 2014版性能到底怎么样还要看实际使用，对于日常应用和小游戏一般的处理器都能轻松胜任，性能强劲的Exynos 5420处理器自然流畅运行，因此笔者直接略过，进入比较流行的大型3D游戏测试。

　
地铁跑酷

　　地铁跑酷只是用来体现三星Galaxy Note 10.1 2014版屏幕超级赞的显示效果，接下来笔者下载了几款有数据包的大型游戏，完全可以流畅运行，没有丝毫卡顿，双扬声器立体声效果也很好。


激流快艇2

　　笔者试玩了激流快艇2、猎鹿人2014、杀手2等几款游戏，表现都十分出色，完美运行且丝毫没有卡顿，加载也非常快，特别是激流快艇2还有水花飞溅的效果，要知道这款游戏是专为英伟达Tegra 3打造的。


猎鹿人2014

　　笔者认为这些游戏依然不能发掘Exynos 5420处理器、3GB运行内存的全部实力，这样的配置在现在真的有些“产能过剩”，如果你只用这款平板刷微博、浏览网页、看视频的话，真的有些暴殄天物了。


杀手2

兼容绝大多数视频格式

　　人们使用平板电脑很大一部分时间用来观看高清视频，对于Note 10.1 2014版这样的大尺寸平板电脑来说，娱乐影音功能必然是最看重的。


视频播放测试

　　三星Galaxy Note 10.1 2014版在视频方面的性能确实很强悍，视频播放器有浮窗预览功能，触控笔距离视频一定距离就会显示显示文件名、视频时长、已播放时间等相关信息，非常实用。


视频播放测试

　　从支持的格式来看，三星Galaxy Note 10.1 2014版基本上做到了全格式播放，AVI、MKV、MOV、MP4、RMVB、VOB、WMV、3GP等主流视频都可以直接播放，无需转码或者第三方解码。


视频播放测试

　　高清视频播放完全是这颗Exynos 5420处理器的功劳，播放720p视频毫无压力，即使是1080p影片，也可以即时载入，拖动进度条、快进、快退都没有延迟和卡顿，辅以2560×1600超高分辨率屏幕和双扬声器，观看视频的感觉怎一个“爽”字了得。

　　总结：今年的GALAXY Note 10.1 2014版要比去年的机型贵，从硬件来说，三星的确带来了不少亮点：设备的机身更轻更薄，设计看上去更加干净，配置也有所提升，屏幕的显示效果也是一流，但三星的平板是否会对苹果iPad系列造成威胁还需要时间的检验，但作为目前Andorid系统的霸主平板，三星还有不少路要走。
