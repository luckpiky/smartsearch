8寸平板谁最强?三星Galaxy Note 8.0评测
site name  2013-11-04
原文地址：http://pad.pconline.com.cn/326/3267644.html

　　【PConline 评测】三星Galaxy Note 8.0自从在MWC亮相之后，人们对它的的关注和评论就没有中断过。有人对它期望颇高，认为它可以与iPad mini一争高下；有人对它诟病颇多，认为它的外观毫无新意，且屏幕分辨率太低。但是谁都无法否认，三星Galaxy Note 8.0成功吸引了业内所有人的好奇心，每个人都想第一时间一探其真容。

　　三星Galaxy Note 8.0分为两个版本，WiFi版和3G/WiFi版，这两个版本均会在国内上市，售价分别为2988元和3688元。相对于其主要竞争对手苹果iPad mini相同内存容量的WiFi版和3G/WiFi版售价要高了不少。不过Note 8.0的3G版是支持通话功能的，有了它之后整个系统的使用体验都发生了改变。

　　顾名思义，Note 8.0的屏幕自然是8.0英寸了，分辨率为1280×800，PPI为189，屏幕材质为TFT。除此之外还搭配了1.6GHz主频的四核Exynos 4412处理芯片以及2GB RAM，这些都与该机准旗舰机型的地位相符。另外在机身内置16GB存储空间的基础上还支持MicroSD卡扩展。


三星Galaxy Note 8.0

外观手感

　　三星最近推出的Galaxy系列机型有着统一的外形风格，基本上都延续了Galaxy三代的鹅卵石造型，这样做的好处是符合了产品的“三米法则”，就算是屏幕上方没有那个“SAMSUNG”的LOGO，即使不通过近距离的观察，消费者也够从远处判断出它是一款来自三星家的产品。不过这些产品一成不变的造型也引起了部分消费者的不满。


Galaxy Note 8.0

　　三星Note 8.0也有着同样风格的外形，第一眼看上去就像是放大了的Note 2。135.9mm×210.8mm×7.95mm的尺，使得一只手刚好可以很舒服的握住。除屏幕外，基本上在Note 8.0的机身上找不到任何直线或者直角的地方，三星对所有的边框和棱角都做了圆滑或弧度的处理，配合上340g的重量，无论我以任何的角度或者方式去握持这款机器，都能获得不错的手感。


弧形边角

　　也许有的人会质疑，Note 8.0为什么没有采用超窄边框设计，采用这种设计可以使得Note 8.0的体积更小，并且看上去更酷。最初我也有过这样的想法，但是当我使用过了几款采用窄边框设计的平板之后，我发现这些窄边框只是看上去很美，由于Android系统并没有类似于IOS系统的防误触设计，使用的时候手只能掐在平板的两边，毫无手感可言。


宽边框设计

　　至于饱受外界批评的塑料材质的机身，我想这是一个见仁见智的问题了，至少笔者并没有觉得手中的Note 8.0有任何廉价感，白色的钢琴钢琴烤漆让其看起来十分的典雅，虽然这种涂层比较容易沾染指纹什么的在机身表面，但是背部点阵样式的花纹可以很好的将指纹掩盖掉，所以消费者也不用过于担心它会影响美观的问题。


机身背面

　　除去体积，材质以及重量，另一个影响手感的因素的就是机身发热，经过测试，笔者认为日常使用过程中Note 8.0对于发热量的控制还是很好的，轻度和中度使用过程中，基本感觉不到机身热量，如果是长时间运行大型游戏或者应用的话，可以感觉到机身左侧有明显的发热，这正好是左手握持的部位，对于日常使用还是有一些影响的，还有就是充电过程中机身左侧的发热量也十分的明显，还好Note 8.0的充电速度很快，基本两个半小时左右便可以充满，所以充电的时候还是不要使用的好。

系统配置

　　三星Note8.0的硬件信息在发布之前就早已曝光，整机配置达到了当前平板电脑的顶级水平。虽然没有配备三星自家最新推出的Exynos 5 Octa八核处理器，但是其内置的Exynos 4412四核处理器目前看性能仍然足够强大，配合上2G超大内存，无论跑分还是日常使用表现均十分优秀。此外Note8.0还配备了8寸1280×800的高清TFT屏幕，内置了16GB的存储容量并支持MicroSD卡扩展。

　　三星Note 8.0在安兔兔测试软件评测中取得了17141的高分，目前市面上的其他四核平板没有能与之匹敌的，只是与三星最新推出的Exynos 5 Octa八核处理器的得分有着30%左右的差距，但在实际使用过程中差别并没有分数上的那么明显。


安兔兔跑分

　　虽然此次三星Note 8.0没有采用自家招牌的Super AMOLED屏幕，但显示效果依然有着较高水准，屏幕饱和度很高，色彩显示保持了三星一贯浓郁鲜艳的风格。


屏幕色彩显示十分艳丽

　　由于笔者拿到的Note 8.0是WiFi版，所以机身的顶部并没有配备听筒，只是配置了前置摄像头和光线感应器，这枚前置摄像头的像素为130万，成像质量令人满意，完全可以满足日常视频聊天的需求，偶尔还可以自拍一下。除前置摄像头以外，三星Note 8.0也配备了一枚500万像素的后置摄像头，遗憾的是并没有配置闪光灯，不过笔者认为也没有人会将平板作为主力照相设备吧。令我感到意外的是这枚摄像头是采用突出式设计，估计这会让那些外观控们纠结上一番，平日使用时也要注意防止要刮花。


前置摄像头及光线感应器


后置摄像头

　　三星Note8.0搭载的系统为Android4.1.2，并同Galaxy家族的其他机型一样，搭载了最新的三星自己深度定制的TouchWiz操作界面，与之前版本的TouchWiz界面相比，这一代的TouchWiz界面更多的采用了明亮的色彩搭配，系统框架与一些元素搭配都使用了随和的圆角矩形样式，配合上三星自带的壁纸，整个系统的风格给人一种很清新自然的感觉。


操作界面

　　除了图形界面层面的改变以外，三星同样对Android系统本身做了很大的修改，例如增加了对S Pen和手部动作支持，将谷歌的网络服务替换为自己的云服务，这些系统层面的更改一方面为用户使用增加了便利，另一方也体现出三星并不甘心屈服在谷歌的控制之下，只是做一个设备生产商，而是要培养三星自己的用户习惯以及品牌效应。


三星内置服务

　　目前已经有国外媒体报道，三星Galaxy品牌的影响力已经超越了Android品牌本身。当然，不系统如何更改，只有方便易用的系统才能得到用户的选择。

按键接口

　　Note 8.0采用了Android平板上少有的实体按键设计，一枚实体的Home键以及另外两枚触摸式按键，分别为菜单键和返回键。这样的设计一方面避免了屏幕可视面积的浪费，另一方面也保留了原有Galaxy系列的操作习惯，减少了用户学习的成本。不过由于按键占据屏幕底部边框的大部分面积，横向握持的时候要注意一下，防止无意间碰触造成误操作。


实体按键

　　三星Note 8.0总共设计有4个接口，一个是在顶端的耳机接口，一个是位于底部的Micro USB数据接口，还有就是位于机身左侧的内存卡扩展槽。另外一个下文会说到。

　　三星将音量键和电源键设计在了机身右侧，这样的设计十分符合用户日常的使用习惯，无论左手还是右手单手握持，都可以很方便的使用这些按键。此外这里还有Note 8.0的第四个接口——红外线接口，通过内置的智能遥控程序，Note 8.0可以瞬间变身成万能遥控器，不仅可以与三星自家的电视相匹配的，还可控制其他主流品牌的电视。


机身右侧的按键及红外接口

S Pen多功能笔

　　作为Galaxy Note 家族的一名成员，Note8.0肯定少不了多功能手写笔S Pen，外观上它与Note 家族中其他成员所配备的S Pen形状类似，触笔的顶端采用了与Note 8.0外壳表面相同的钢琴烤漆，笔杆部分则是使用了更能够增加其与使用者手指之间摩擦力的磨砂处理。尺寸方面，这根触笔的长度大约与正常男性用户的手掌宽度差不多，笔杆相对于NOTE 10.1上的要更细一些。在具体使用过程中，感觉这根Note 8.0上的S Pen握持起来还是挺舒服，便于用户在书写时发力。


S Pen

　　Note 8.0的触笔操位于机身的右下角，用户可以很轻松的利用触笔顶端与插槽中间的凹槽空隙将手写笔取出来。取出同时屏幕的左上角也会弹出一个触笔形状的图标，这样的设计可以有效的避免使用者在使用完S Pen触笔之后忘记把它插回到插槽之中，减少了它被遗失的概率。为了防止在使用spen过程中手掌等部位接触屏幕产生干扰，当我们拔出手写触笔时，Note 8.0也会自动通过检测手写笔的位置，当笔尖距离屏幕14mm之内时，系统会自动出发防误触功能，另外在书写过程中，由于笔端加入了胶粒，因此更有利于使用者对与这个触笔在屏幕上滑动距离的控制。而压感方面，Galaxy Note 8.0的S Pen触笔和Galaxy Note 10.1的一样，同为1024级。


机身右侧的S Pen笔槽


S Pen顶端胶粒

　　三星为了让用户能够更好地使用这只S Pen，围绕着它开发了许多的应用，并且在系统中也针对S Pen做了相关的优化。借助S Pen我们可以快捷的完成各种操作和实现各种功能。

S Note模式

　　我们如果在待机情况抽出S Pen，系统会自动进入S Pen模式并自动打开S Note应用，此时我们可以很方便的根据自己的需求，在S Note中选取不同的文本模式的。每种文本模式都做好了相应的设置，方便我们插入图片，视频或者进行涂鸦。此外，S Pen模式下的系统通知栏里也内置了相应的snote快捷方式和快捷命令方便用户使用。


S Pen模式

提笔速记

　　如果遇到紧急情况时，Note 8.0还提供了锁屏模式下抽出触笔自动激活S Note应用的功能，大大提高了整套操作流程的执行效率。此外通过按住S Pen触笔笔杆上的侧向功能键，然后在屏幕上快速双击两下也可以快速激活S Note。


提速笔记

屏幕截图

　　通过S Pen我们还可以获取任意形状的屏幕截图。要实现此项功能，用户需要在按住笔杆上的功能键后使用笔尖在Note 8.0的屏幕上画一个闭合的图形，当闭合图形画完之后这部分内容就就会被截取下来，并且会自动打开图片编辑器，使用者可以为它添加一些批注，或者是直接拖拽到S Note应用中作为素材使用。


屏幕截图

浮窗预览

　　当S Pen电容触笔距离屏幕较近时，屏幕上会根据位置感应信息在屏幕上绘制一个与S Pen笔尖所对应的像素点，它就相当于电脑屏幕上的鼠标指针。那么我们都知道鼠标指针具有悬停预览功能，同样的Note 8.0也支持S Pen触笔悬停预览，当我们把S Pen触笔的笔尖悬在某个屏幕中的元素上时，就会浮出一个窗口显示与这个元素相关的信息。


浮窗预览

续航时间　

　　三星Note 8.0虽然配备了4600mAh的电池，但是由于搭载了高性能的CPU和8寸的高清屏幕，对电池有很大的压力，而且当电量低于10%的时候，Note 8.0会自动进入低电模式，此时屏幕的亮度将调到最低，基本上无法正常使用。


低电模式

　　此外，笔者对手中的Note 8.0进行了极限使用测试，Note 8.0可以连续玩《极品飞车 17》这样的大型3D游戏四个小时左右（开启WiFi连接、音量大小与背光亮度均调整为最大值的一半、开启通知），而Note 8.0在视频播放方面则有着十分优异的表现，可以连续720P高清视频达到10小时以上。

使用感受


Galaxy Note 8.0

　　笔者此前也使用过Galaxy Note 10.1和Galaxy Note 2，但是此次Galaxy Note 8.0给我带来了完全不同的使用感受，Note 10.1被三星官方定义为生产力工具，虽然拥有10.1英寸的大屏幕，但是由于尺寸以及重量的因素，便携性上并不能令我满意； Note 2虽然体积小巧，但是其大部分时间还是以手机的身份存在，且作为平板来讲屏幕也太小，而Note 8.0则在这两者之间取得了一个刚刚好的平衡，大小和重量与一个纸质的记事本没有差别，随身携带毫无负担，而8寸的屏幕可以很方便的进行各种操作，尤其是融入了通话功能，将平板Galaxy Note 8.0带入了一个新的层次。
